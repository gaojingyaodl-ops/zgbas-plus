## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 外部 xxl-job admin DB → 导出物 | 用户从生产 admin DB 导出 xxl_job_info 表；导出物可能含生产敏感 cron 节奏 / 业务参数 / handler 路由策略 |
| 导出物 → 翻译产物 | Claude 翻译过程引入翻译错误风险（cron 字段错配 / invoke_target 误构 / status 分级误判）|

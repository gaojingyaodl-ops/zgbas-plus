<!-- WxConfiguration 关键结构 -->
来自源文件 WxConfiguration.java（照搬，仅换 package）：
- 注解：@Configuration @EnableConfigurationProperties(WxMiniAppConfig.class)
- static 字段：Map&lt;String, WxMaService&gt; maServices / Map&lt;String, WxMaMessageRouter&gt; routers
- 构造器：@Autowired WxConfiguration(WxMiniAppConfig properties) — 将 properties 赋给 static 字段
- static getMaService(String appid) + static getMaService()（用 properties.getConfigs().get(0).getAppid()）
- static getRouter(String appid)
- @PostConstruct init()：遍历 configs 列表，为每个 config 创建 WxMaDefaultConfigImpl + WxMaServiceImpl
- private final WxMaMessageHandler 字段（logHandler/textHandler/picHandler/qrcodeHandler/templateMsgHandler）

BaseException 依赖：仅在 init() 中被抛出：throw new BaseException(Status.ERROR, "小程序配置错误")

<!-- WxMiniAppConfig 结构（照搬无改动） -->
- @Data @ConfigurationProperties(prefix = "wx.miniapp")
- List&lt;Config&gt; configs
- static inner @Data class Config { appid, secret, token, aesKey, msgDataFormat }

<!-- Common stubs 需包含的最小内容 -->
Status enum 需包含：ERROR(500, "操作异常！")，以及 04-04 JwtUtil 使用的：
  TOKEN_EXPIRED(5003, ...), TOKEN_PARSE_ERROR(5004, ...), TOKEN_OUT_OF_CTRL(5005, ...),
  NO_ACCESS_TOKEN(5010, ...), UNAUTHORIZED(401, ...), SUCCESS(200, ...)
可直接照搬源码完整 Status.java（便于 Phase 5 直接引用而无需修改）。

BaseException：extends RuntimeException，含 Integer code / String message / Object data 字段，
  构造器：BaseException(Status status) + BaseException(Status status, Object data) +
         BaseException(Integer code, String message) + BaseException(Status status, String message)

ApiResult：ResponseUtil 需要 ApiResult.ofStatus(status, data) 静态工厂方法（04-04 Task 中 ResponseUtil 创建时用到）。
  照搬源码即可（com.spt.bas.purchase.wx.server.common.ApiResult）。

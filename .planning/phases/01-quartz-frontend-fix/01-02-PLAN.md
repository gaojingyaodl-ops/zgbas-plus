### add.html 当前问题 (line 82-84)
```javascript

    $.operate.save(prefix + "/add", $('#form-job-add').serialize());

```
- prefix = "ctx + monitor/job"（即 "/monitor/job"）
- URL 错误：prefix + "/add" = "/monitor/job/add"（应为 "/monitor/job"）
- data 错误：.serialize() 产生 form-encoded 字符串（应为 JSON）
- 需改为：$.ajax POST + contentType: application/json + JSON.stringify(data)

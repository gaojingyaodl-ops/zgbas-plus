新方法代码（复制 post 的结构，但使用 put + JSON body）：
```
            put: function(url, data, callback) {
                $.ajax({
                    url: url,
                    type: "put",
                    dataType: "json",
                    contentType: "application/json;charset=utf-8",
                    data: JSON.stringify(data),
                    dataType: "json",
                    beforeSend: function () {
                        $.modal.loading("正在处理中，请稍候...");
                    },
                    success: function(result) {
                        if (typeof callback == "function") {
                            callback(result);
                        }
                        $.operate.ajaxSuccess(result);
                    }
                });
            },
```

关键要点：
  - 直接调用 $.ajax（不通过 $.operate.submit）因为需要设置 contentType
  - contentType: "application/json;charset=utf-8" 确保 Controller 的 @RequestBody SysJob 能正确反序列化
  - data: JSON.stringify(data) 将 JS 对象序列化为 JSON 字符串
  - type: "put"（不是 "post"）
  - success 调用 $.operate.ajaxSuccess(result) 保持与现有 $.operate 统一的刷新 + 弹出 + loading 关闭行为
  - 其他字段从 post 方法复制（beforeSend loading, dataType "json", callback 原样转发）

不要修改 $.operate.submit、$.operate.post 或任何其他现有方法。只添加新方法。

```
function removeJob(jobId) {
    $.modal.confirm("确定删除该条定时任务吗？", function() {
        $.ajax({
            url: prefix + "/" + jobId,
            type: "delete",
            dataType: "json",
            beforeSend: function () {
                $.modal.loading("正在处理中，请稍候...");
            },
            success: function(result) {
                if (result.code == 0) {
                    $.modal.msgSuccess(result.msg);
                    $.table.refresh();
                } else {
                    $.modal.alertError(result.msg);
                }
                $.modal.closeLoading();
            }
        });
    });
}

function removeAllJobs() {
    $.table.set();
    var rows = $.table.selectFirstColumns();
    if (rows.length == 0) {
        $.modal.alertWarning("请至少选择一条记录");
        return;
    }
    $.modal.confirm("确定删除选中的" + rows.length + "条任务吗？", function() {
        $.ajax({
            url: prefix + "/" + rows.join(","),
            type: "delete",
            dataType: "json",
            beforeSend: function () {
                $.modal.loading("正在处理中，请稍候...");
            },
            success: function(result) {
                if (result.code == 0) {
                    $.modal.msgSuccess(result.msg);
                    $.table.refresh();
                } else {
                    $.modal.alertError(result.msg);
                }
                $.modal.closeLoading();
            }
        });
    });
}
```

实现要点：
  - removeJob: DELETE {prefix}/{jobId} → Spring MVC 匹配 @DeleteMapping("/{jobIds}")，Long[] jobIds = [jobId]
  - removeAllJobs: DELETE {prefix}/{id1,id2,...} → Spring MVC 匹配 @DeleteMapping("/{jobIds}")，Long[] jobIds = [id1, id2, ...]
  - 手动处理 result.code == 0 判断 + $.table.refresh()（因为 $.operate.ajaxSuccess 依赖 table.options.type 上下文，简化为直调更可靠）
  - startJob/pauseJob 函数不变 — $.operate.put 已由 Task 1 添加，现有调用自动生效
